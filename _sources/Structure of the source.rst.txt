Structure of the source
-----------------------

The following the structure of Schemey's repo:

* ``doc\``: This directory contains all of the documentation for Schemey.
* ``src\``: This directory contains all of the source files created and/or used in
            Schemey
* ``tests\``: This directory contains all of the Scheme source files used to
              test the functionality of Schemey.
* ``README.md``: This file contains the README of this project.
* ``UNLICENSE.txt``: This file contains a copy of the *unlicense* license(See
                     *License* for more details).
